# Practica_25VVV3_7
Учебная практика. Двоичная сортировка (binary tree sort) на C++. Вариант 7.
Бригада: Радушев И.А. (логика), Ратьков Д.И. (интерфейс), Китаев И.А. (тестирование).
Структура:
praktice_1/ — проект Visual Studio (.sln + исходный код).
tests/ — тестовые наборы данных.
bin/ — готовый praktice_1.exe для запуска без сборки.
Файлы проекта
practice_1.slnx — файл решения Visual Studio. Открывается для загрузки всего проекта.
practice_1.vcxproj — файл проекта: настройки сборки, список исходников.
practice_1.vcxproj.filters — распределение файлов по группам в дереве проекта.
main.cpp — точка входа: меню и взаимодействие с пользователем.
bst_sort.h / bst_sort.cpp — алгоритм двоичной сортировки: построение и обход дерева, подсчёт операций.
io.h / io.cpp — чтение и запись файлов, генерация случайного массива.
benchmark.h / benchmark.cpp — анализ производительности на отсортированном, инвертированном и случайном наборах.
Варианты запуска:
Через Visual Studio: открыть Praktika/Praktika.sln.
Без сборки: запустить bin/Praktika.exe двойным кликом.
