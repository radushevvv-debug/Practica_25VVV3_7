# Practica_25VVV3_7
Учебная практика. Двоичная сортировка (binary tree sort) на C++. Вариант 7.
Бригада: Радушев И.А. (логика), Ратьков Д.И. (интерфейс), Китаев И.А. (тестирование).
Структура:
praktice_1/ — проект Visual Studio (.sln + исходный код).
tests/ — тестовые наборы данных.
bin/ — готовый praktice_1.exe для запуска без сборки.

Варианты запуска:
Через Visual Studio: открыть Praktika/Praktika.sln.
Без сборки: запустить bin/Praktika.exe двойным кликом.
